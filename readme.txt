An AI-powered email assistant plugin for WordPress.

== Description ==

SmartMail Assistant is an AI-powered email assistant plugin that enhances productivity by automating and optimizing email management. It offers features like email categorization, priority inbox, automated responses, email summarization, meeting scheduler, follow-up reminders, sentiment analysis, and email templates.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/smartmail-assistant` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->SmartMail Assistant screen to configure the plugin.
